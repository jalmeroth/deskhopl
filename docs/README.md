## GPIO/Pins

### PICO A

| GPIO | Description | optional |
| ---- | ----------- | :------: |
| 4    | UART1 TX    |    x     |
| 5    | UART1 RX    |    x     |
| 12   | UART0 TX    |          |
| 13   | UART0 RX    |          |
| 14   | USB DP PIN  |          |

### PICO B

| GPIO | Description | optional |
| ---- | ----------- | :------: |
| 4    | UART1 TX    |    x     |
| 5    | UART1 RX    |    x     |
| 14   | USB DP PIN  |          |
| 16   | UART0 TX    |          |
| 17   | UART0 RX    |          |
